🧹 Pearcleaner: A free open-source application cleaning tool for macOS
{{< /side-line >}}

On macOS, many applications leave behind residual files after deletion, occupying disk space and potentially affecting system performance. Pearcleaner is a free and open-source application cleaning tool designed to help users completely remove applications and related files, keeping their systems clean.

<br>

{{< side-line color="#fe53bb" >}}
🔍 Main Features
{{< /side-line >}}

* Isolated File Search: Able to scan and identify residual files left by deleted applications, assisting users in thoroughly removing unnecessary information.
* Development Environment Cleanup: Provides cleaning functionalities for temporary files and snapshots generated in the development environment, helping developers maintain a clean working environment.
* App Lipo Features: Remove unnecessary architecture from Universal applications, reduce their size, and do not rely on Xcode or command-line tools.
* Sentinel Monitoring Assistant: A lightweight background tool (only 210KB, approximately 2MB of RAM) that monitors application deletion behavior in the trash can and prompts users to clean up any remaining files.

<br>

{{< side-line color="#fe53bb" >}}
🧪 Technical Details and Authorization
{{< /side-line >}}

* Developed in Swift: The entire application is written in Swift, providing a modern and native macOS experience. 
* Authorization method: Using Apache 2.0 with Commons Clause licensing allows for free use, modification, and contribution, but prohibits the commercialization of the original or modified versions
